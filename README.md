# Personal Firewall

## Overview

This project is an intermediate Python-based personal firewall developed as part of a Cyber Security Internship.

## Features

- Block IP addresses
- Block ports
- Validate IP addresses
- Log all connection attempts
- JSON-based configuration
- Simple command-line interface

## Technologies

- Python 3
- JSON
- Logging Module
- ipaddress Module

## Project Structure

firewall.py
rules.py
logger.py
config.json
logs/
README.md

## Run

python3 firewall.py
